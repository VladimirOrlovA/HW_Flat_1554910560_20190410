Разработать класс Flat (квартира) на основе класса Room(комната). Класс  Flat содержит поле цена и количество комнат(массив). Класс должен содержать: конструктор по умолчанию, конструктор с параметрами, при необходимости реализовать деструктор и конструктор копирования.
Добавить методы для:
□	определения  площади квартиры (сумма площадей всех комнат)
□	метод для проверки относятся ли квартиры  к одному типу (одно, двух, трехкомнатная)
□	для проверки равенства площадей  квартир
□	для сравнения цены квартир
□	для копирования объектов
□	остальные методы на усмотрение разработчика (методы set и get)
