Разработать класс Flat (квартира) на основе класса Room(комната). Класс  Flat содержит поле цена и количество комнат(массив). Класс должен содержать: конструктор по умолчанию, конструктор с параметрами, при необходимости реализовать деструктор и конструктор копирования.
Добавить методы для:
ok - □	определения  площади квартиры (сумма площадей всех комнат)
ok - □	метод для проверки относятся ли квартиры  к одному типу (одно, двух, трехкомнатная)
ok - □	для проверки равенства площадей  квартир
ok - □	для сравнения цены квартир
ok - □	для копирования объектов
ok - □	остальные методы на усмотрение разработчика (методы set и get)
